# jsTreeRails

Makes including jsTree in Rails 3.1 projects a breeze.

## Installation

Add the following line to your Gemfile:

    gem 'jstree-rails', :git => 'git@github.com:tristanm/jstree-rails.git'

then run `bundle install`
